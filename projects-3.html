File Hiding Tool
This tool was adding a file to another file to hide it. This was like kind of another security method. It is hard to guess the presence of a file within another file if you have never encountered this method before. Also, it is not easy to split merged files, easily. You could carry a PDF file in an executable file. Later, you could split them by using this software again.

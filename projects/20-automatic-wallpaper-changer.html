Automatic Wallpaper Changer
This was a unique software in 2007. Later, changing wallpaper automatically has become so popular. Even mobile phones had this feature. It is a built-in Windows feature today. This software was using your selected image folder and it was changing your wallpaper according to your time settings. It was supporting bmp, jpg, gif, png, tiff formats and quietly working in the system tray.
